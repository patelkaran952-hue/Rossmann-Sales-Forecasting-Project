🛒 Rossmann Sales Forecasting & Business Intelligence Project

📌 Project Overview

This project focuses on analyzing and forecasting sales for the Rossmann retail chain using data analytics, machine learning, statistical analysis, and business intelligence tools.

The objective was to transform raw retail data into actionable business insights and accurate sales forecasts through an end-to-end data science workflow.


🚀 Project Workflow

1️⃣ Data Cleaning & Preprocessing (Python)

Performed comprehensive data cleaning and preprocessing using Python.
